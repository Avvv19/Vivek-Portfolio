## Overview

My **personal portfolio website** built with Next.js, TypeScript, and Tailwind CSS — showcasing my journey as an AI in Healthcare professional. The site highlights my projects, skills, experience, and passion for using artificial intelligence to transform healthcare outcomes.

---

## Features

- **Responsive Design** — Mobile-first, works on all screen sizes
- **Project Showcase** — Interactive cards for all AI/ML projects
- **Skills Section** — Animated tech stack visualization
- **About & Experience** — Professional timeline and achievements
- **Contact Form** — Direct reach for collaboration opportunities
- **Dark Mode** — Clean dark/light theme toggle
- **Performance Optimized** — 90+ Lighthouse score

## Getting Started

```bash
# Clone the repository
git clone https://github.com/Avvv19/Vivek-Portfolio.git
cd Vivek-Portfolio

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
